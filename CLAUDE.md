## 화면 정의

### 가계부 (v0.1)

#### 메인 화면 — 뷰 전환 탭 3개

상단 탭으로 3가지 뷰 전환:

| 탭 | 설명 |
|----|------|
| 달력 | 월간 캘린더. 날짜별 총지출 + 낭비금액 표시. 월 이동 가능 |
| 주차별 | 현재 주차 지출 목록. 좌우 화살표로 주차 이동 |
| 전체 로그 | 전체 기간 지출 최신순 나열. 무한 스크롤 |

#### 달력 뷰
- 월 이동 (◀ ▶)
- 날짜 셀: 총지출 + 낭비금액 + 건수 표시 (데스크탑) / 컬러 점 (모바일)
- 날짜 클릭 → 일별 상세 모달

#### 주차별 뷰
- 현재 주차 데이터만 표시 (기본: 가장 최근 주차)
- 좌우 화살표로 주차 이동 (◀ 5월 2주차 ▶)
- 날짜 카드: 날짜+요일 헤더 / 지출 목록 / 일별 합계 푸터
- 낭비 항목: ⚠️ 아이콘 + 주황색 표시
- 항목 클릭 → 일별 상세 모달

#### 전체 로그 뷰
- 전체 기간 날짜별 최신순 나열
- IntersectionObserver 무한 스크롤 (5개 날짜씩 추가 로딩)
- 날짜 카드 구조 주차별 뷰와 동일

#### 사이드바 (데스크탑)
- 요약 카드 4개: 이번달 월급 / 총지출 / 남은금액 / 현금통장잔액
- 이번달 낭비 금액 + 지출 대비 비율
- 카테고리별 지출 바 차트 (상위 5개)

#### 일별 상세 모달
- 상단: 날짜 + 닫기 버튼
- 총지출 / 낭비 요약 카드
- 지출 목록 (금액 + 사용처 + 카테고리 + 낭비 표시)
- 낭비 항목: 아이콘 + 주황색 금액으로 구분
- 하단: 지출 추가 버튼

#### 지출 입력 모달
- 날짜 선택
- 금액 입력
- 사용처 입력
- 카테고리 선택 (확장 가능한 구조)
- 낭비 여부 토글 (수동 입력)
- 저장 버튼
