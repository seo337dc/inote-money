## 프로젝트 개요

- **한줄 소개**: 개인 수입/지출, 투자 포트폴리오를 한 곳에서 관리하는 웹/앱 서비스
- **배경/문제**: 가계부·투자 기록이 분산되어 있어 전체 자산 현황을 한눈에 파악하기 어려움
- **목표/성공 기준**: 매일 지출 입력 → 월말 리뷰 → 낭비 줄이기 습관 형성

## 화면 정의

### 가계부 (v0.1)

#### 메인 화면 — 뷰 전환 탭 3개

상단 탭으로 3가지 뷰 전환:

| 탭 | 설명 |
|----|------|
| 달력 | 월간 캘린더. 날짜별 총지출 + 낭비금액 표시. 월 이동 가능 |
| 주차별 | 현재 주차 지출 목록. 좌우 화살표로 주차 이동 |
| 전체 로그 | 전체 기간 지출 최신순 나열. 무한 스크롤 |

#### 달력 뷰
- 월 이동 (◀ ▶)
- 날짜 셀: 총지출 + 낭비금액 + 건수 표시 (데스크탑) / 컬러 점 (모바일)
- 날짜 클릭 → 일별 상세 모달

#### 주차별 뷰
- 현재 주차 데이터만 표시 (기본: 가장 최근 주차)
- 좌우 화살표로 주차 이동 (◀ 5월 2주차 ▶)
- 오늘 카드 항상 최상단 표시 (데이터 유무 무관)
- 날짜 카드: 날짜+요일 헤더 / 지출 목록 / 카테고리 뱃지 / 일별 합계 푸터
- 낭비 항목: ⚠️ 아이콘 + 주황색 표시
- 카드 클릭 → 인라인 입력 폼 열림 (추가/수정 가능)
- 저장 버튼으로 폼 닫기

#### 전체 로그 뷰
- 전체 기간 날짜별 최신순 나열 (오늘 카드 최상단 고정)
- IntersectionObserver 무한 스크롤 (5개 날짜씩 추가 로딩)
- 날짜 카드 구조 및 동작 주차별 뷰와 동일 (클릭 시 인라인 폼)

#### 사이드바 (데스크탑)
- 요약 카드 4개: 이번달 월급 / 총지출 / 남은금액 / 현금통장잔액
- 이번달 낭비 금액 + 지출 대비 비율
- 카테고리별 지출 바 차트 (상위 5개)

#### 일별 상세 모달 (달력 뷰 전용)
- 상단: 날짜 + 닫기 버튼
- 총지출 / 낭비 요약 카드
- 지출 목록: 금액 → [⚠️] → 사용처 → [카테고리 뱃지]
- 낭비 항목: ⚠️ 아이콘 + 주황색 금액으로 구분
- 하단: 지출 추가 버튼

#### 지출 인라인 입력 폼 (주차별 / 전체 로그)
- 금액 입력 (숫자, 필수)
- 사용처 입력 (텍스트, 선택)
- 낭비 토글
- 카테고리 칩 선택
- 추가 버튼 → 즉시 리스트 반영, 폼 초기화 후 유지 (연속 입력)
- 저장 버튼 → 폼 닫기

### 대시보드 (데모 완료)

- **내 정보 카드**: 월급 / 적금 합계 / 고정지출 합계 / 일일 한도 요약, 상세 펼치기
- **주간 리뷰 카드**: 주 이동, 이번주 지출/낭비, 별점(1~5) + 리뷰 텍스트, localStorage 저장
- **월간 요약 카드**: 월 이동, 총지출/낭비/저축 목표, 별점 + 리뷰 텍스트, localStorage 저장

### 내 정보 설정 (데모 완료)

- 월급 + 입금일 / 일일 한도 / 월 저축 목표 / 자산 업데이트일
- 적금 목록: 이름 + 금액 + 이체일 (다중 추가/삭제)
- 고정 지출 목록: 이름 + 금액 + 이체일 (다중 추가/삭제)
- localStorage(`inote-settings`) 저장

### 주식 (데모 완료)

**국내 주식 / ETF 섹션**
- 가로 스크롤 카드 (수량 기반: 보유주수 × 매입가)
- 차트: Naver 금융 fchart API → Next.js API Route 프록시 → Lightweight Charts v5 캔들스틱
- ISA ETF(TIGER S&P500, 나스닥100, 미국우주테크) 지원

**해외 주식 섹션**
- 가로 스크롤 카드 (금액 기반: 총 투자금액 달러)
- 차트: TradingView iframe embed

**환율**
- USD/KRW, 100엔/KRW, EUR/KRW — open.er-api.com (무료, 키 불필요)
- 새로고침 버튼

**입력 방식 구분**
- 수량 모드 (국내 기본): 수량(주) + 평균 매입가
- 금액 모드 (해외 기본): 투자 금액(USD)만 입력

## 정책 (Policy)

### 용어 정의

- **낭비**: 사용자가 주관적으로 판단하는 불필요한 지출. 자동 분류 없이 수동 토글로 처리
- **카테고리**: 식비 / 카페 / 교통 / 쇼핑 / 의료 / 문화 / 구독 / 기타

### 데이터 정책

- 삭제: 소프트 삭제 (추후 결정)
- 더미 데이터: `/demo` 경로에서만 사용, 새로고침 시 초기화

### 에러/예외 정책

- 금액 미입력 시 추가 버튼 비활성화
- 사용처 미입력 시 `-` 로 저장

### 권한/역할

- 1인 개인 서비스 → 별도 권한 구분 없음
