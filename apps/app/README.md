# apps/app — iNote Money Mobile

> iNote Money 모바일 앱 — React Native _(예정)_

---

## 상태

🚧 **개발 예정**

웹 서비스(`apps/web`) 데모 화면 완성 후 모바일 앱 개발을 시작할 예정입니다.

---

## 예정 기술 스택

| 항목 | 기술 |
|------|------|
| 프레임워크 | React Native (Expo) |
| 상태관리 | Zustand 또는 Jotai |
| 스타일 | NativeWind (Tailwind for RN) |
| 네비게이션 | Expo Router |
| 배포 | App Store · Google Play _(검토 중)_ |

---

## 예정 화면

웹 서비스와 동일한 기능을 모바일 UX에 최적화하여 제공:

- 자산 관리 대시보드
- 가계부 (빠른 지출 입력 중심)
- 주식 포트폴리오
- 설정
